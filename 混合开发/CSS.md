CSS 层叠样式

#### 行内样式：

```
 <p style="color:#ff00ff; background-color:#00ffff"> 行内样式 </p>
```

#### 内嵌样式：

```
<head>
   <style>
      p{
         color:bisque;
         background-color: aqua;
         font-size: 20px;             
         line-height: 60px;        
      }     
    </style> 
</head>
```
